# Gamified Goal Tracker PWA

A "Materialized Path" based goal tracking app with Offline-First capabilities and Social features.

## Tech Stack
- **React 19** (Vite) + **TypeScript**
- **TanStack Query** (State Management & Caching)
- **Firebase** (Auth & Firestore)
- **Tailwind CSS** (Styling)

## Setup
1. Copy `.env.example` to `.env.local` and fill in Firebase keys.
2. `npm install`
3. `npm run dev`

## Features
- Recursive Goals/Sub-goals
- Optimistic UI Updates
- Group Sharing
- PWA Installable
